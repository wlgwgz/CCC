## 能调温度计算

### 能调控制对象

能调控制对象的选择如下：

- 若能调控制温度为 **空调总出水温度`TWout2_ac`** 且 处于特殊功能时，“能调控制对象” 为出水

- 若能调控制温度不为 **空调总出水温度`TWout2_ac`** 或 不处于特殊功能，且使用【**缓冲水箱上部温度探头**】或【**缓冲水箱下部温度探头**】时，“能调控制对象” 为回水

其中，特殊功能指的是：制冷试运行、收氟、能力测试、制热试运行、假日离开模式、地暖烘干功能、地暖预热功能

### 能调控制温度

能调控制温度的选择如下（其中，“～”代表不限）：

| 机组运行模式 | 能量调节方式 | 【**缓冲水箱上部温度探头**】 | 【**缓冲水箱下部温度探头**】 | 能调控制温度 |
|---|---|---|---|---|
| 制冷 / 制热 | “单区室温能调” | ～ | ～ | **室内温度`TR`** |
| 制冷 / 制热 | “单区水温能调”<br/>“双区水温能调”<br/>“双区混合能调” | 禁用 | 禁用 | **空调总出水温度`TWout2_ac`** |
| 制冷 / 制热 | “单区水温能调”<br/>“双区水温能调”<br/>“双区混合能调” | 使用 | 禁用 | **缓冲水箱上部温度`TACt1`** |
| 制冷 / 制热 | “单区水温能调”<br/>“双区水温能调”<br/>“双区混合能调” | 禁用 | 使用 | **缓冲水箱下部温度`TACt2`** |
| 制冷 / 制热 | “单区水温能调”<br/>“双区水温能调”<br/>“双区混合能调” | 使用 | 使用 | • **缓冲水箱上部温度`TACt1`**、**缓冲水箱下部温度`TACt2`** 均可用时：AVG(**缓冲水箱上部温度`TACt1`**, **缓冲水箱下部温度`TACt2`**)<br/>• 仅 **缓冲水箱上部温度`TACt1`** 可用时：**缓冲水箱上部温度`TACt1`**<br/>• 仅 **缓冲水箱下部温度`TACt2`** 可用时：**缓冲水箱下部温度`TACt2`** |
| 制热水 | “热水能调” | ～ | ～ | **热水水箱温度`THWt`** |

其中，**空调总出水温度`TWout2_ac`** 计算方式如下：

- 【**模块个数**】＝ 1或【**热水模块数**】≥【**模块个数**】时，**空调总出水温度`TWout2_ac`** ＝ MODAVG(**总出水温度`TWout2`**)
- 【**模块个数**】＞ 1且【**热水模块数**】＜【**模块个数**】时，**空调总出水温度`TWout2_ac`** ＝ MODAVG_AC(**总出水温度`TWout2`**)

### 能调加载偏差&能调卸载偏差

| 机组运行模式 | 能调控制温度 | 能调加载偏差 | 能调卸载偏差 |
|---|---|---|---|
| 制冷 | **室内温度`TR`** | 【**室温回差**】 | 0 |
| 制冷 | MODAVG(**总出水温度`TWout2`**) | 【**制冷加载偏差**】（若“能调控制对象”为出水则+5℃） | 【**制冷卸载偏差**】 |
| 制冷 | **缓冲水箱上部温度`TACt1`** | 【**制冷加载偏差**】 | 【**制冷卸载偏差**】 |
| 制冷 | **缓冲水箱下部温度`TACt2`** | 【**制冷加载偏差**】 | 【**制冷卸载偏差**】 |
| 制冷 | AVG(**缓冲水箱上部温度`TACt1`**, **缓冲水箱下部温度`TACt2`**) | 【**制冷加载偏差**】 | 【**制冷卸载偏差**】 |
| 制热 | **室内温度`TR`** | 【**室温回差**】 | 0
| 制热 | MODAVG(**总出水温度`TWout2`**) | 【**制热加载偏差**】（若“能调控制对象”为出水则+5℃） | 【**制热卸载偏差**】 |
| 制热 | **缓冲水箱上部温度`TACt1`** | 【**制热加载偏差**】 | 【**制热卸载偏差**】 |
| 制热 | **缓冲水箱下部温度`TACt2`** | 【**制热加载偏差**】 | 【**制热卸载偏差**】 |
| 制热 | AVG(**缓冲水箱上部温度`TACt1`**, **缓冲水箱下部温度`TACt2`**) | 【**制热加载偏差**】 | 【**制热卸载偏差**】 |
| 制热水 | **热水水箱温度`THWt`** | 【**热水回差**】 | 0 |
